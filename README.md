# Ambiguous_emotions_detection


---

## Данные

**GoEmotions** - датасет из 58 000 Reddit-комментариев, размеченных по 27 эмоциям + neutral.

Soft-labels построены из raw-версии датасета на основе долей голосов аннотаторов.

---

## Эксперименты

Сравниваются 4 конфигурации на базе **RoBERTa-base**:

| # | Разметка | Функция потерь|
|---|----------|---------------|
| 1 | One-hot  | BCE           | 
| 2 | Soft     | KL Divergence | 
| 3 | One-hot  | Focal Loss    | 
| 4 | Soft     | Focal Loss    |

---

## Метрики

- **Macro-F1** - основная метрика, показывает качество на редких классах
- **Micro-F1** - качество на частых классах
- **Hamming Loss** - доля неверно предсказанных меток
- **Subset Accuracy** - доля примеров с полностью правильным набором эмоций
